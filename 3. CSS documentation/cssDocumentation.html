CSS basics

We use Cascading Style Sheet (CSS) to style html web pages. We add the following line of code to the .html to link it with the .css file.

<link href="./styles.css" rel="stylesheet">

Selector

A selector allows to target the html elements we want to style. There are different kind:

  • Universal *{property:value;}
  • Type p {CSS declarations;}
  • Id #id {css declarations; }
  • Class .class {css declarations;}
  • Attribute a[attribute=value] { property: value; }
  • Combinators selector1 selector2 / selector 1+selector2 / selector1>selector2 {property: value;}
  • Pseudo selector: pseudo-class{ property: value; }

Box Properties

The CSS box model is a that a rectangular box wraps around each HTML element. These boxes have a border, a padding, a margin and a content. Here are the properties of the box model.

  • Margin margin: value;
  • Padding padding: value;
  • Border border: value;
  • Width width: value;
  • Height height: value;

Media Queries

CSS media queries are used to make web pages more responsive to the viewer's kind of the screen. They are used to check the size, orientation and resolution of the user device. They consist in a CSS block that is apply only if a condition is true or false.

Syntax:

@media not|only mediatype and (expressions) { CSS-Code; }

Media type:

  • All
  • Print
  • Screen
  • Speech
